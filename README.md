# 로또 번호 추천 프로그램
### 개요
> 일반적으로 로또 번호를 고를땐, 사용자가 직접 번호를 관리하거나 마음에 드는 번호가 나올때까지 추천을 하고 싶은 경우가 있습니다.<br>
> 로또 구매시, 자동 추천 기능이 있지만 이는 직접 구매해야 번호를 알 수 있는 문제가 있습니다.<br>
> 그래서 로또를 구매하기 전에 마음에 드는 번호를 뽑을 수 있는 프로그램을 구현함으로써 위 문제를 해결할 수 있습니다.<br>

### 기능
> 중복없이 랜덤한 숫자 6개를 출력합니다.<br>
> 추천후 Y 입력시, 새로운 숫자를 추천해줍니다.<br>
> 추천후 N 입력시, 프로그램을 종료합니다.<br>
