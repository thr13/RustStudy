# 우테코 프리코스 미션
낯선 도구 해커톤: 평소에 잘 사용하지 않는 개발 도구나 언어로 문제 해결하기 그리고 동작하는 작은 결과물 제출

# 로또 번호 추천 프로그램
중복없이 랜덤한 숫자 6개를 출력합니다.<br>
추천후 Y 입력시, 새로운 숫자를 추천해줍니다.<br>
추천후 N 입력시, 프로그램을 종료합니다.<br>
